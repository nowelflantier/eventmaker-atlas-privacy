Privacy Policy

Eventmaker Atlas Overlay · Last updated: April 20, 2026

Overview

Eventmaker Atlas Overlay is a Chrome extension that provides a configuration documentation overlay for the Eventmaker back-office. This policy explains what data the extension accesses, how it is stored, and what it does not do.

Data accessed

The extension accesses the following data:

  • Eventmaker configuration data — the extension makes read-only GET requests to the Eventmaker API (the platform you are already logged into) to retrieve configuration objects such as guest fields and segments. No credentials are collected or stored by the extension; it relies solely on the authentication session already active in your browser.
  • Current page URL — the extension reads the URL of the active Eventmaker page to detect context (which event, which section) and display relevant information.

Data storage

All data is stored locally on your device only:

  • IndexedDB — used to cache Eventmaker configuration data fetched from the API, so the extension works without repeated network calls.
  • localStorage — used to store lightweight usage data such as recently visited events and feature visit counts, to improve navigation within the overlay.

None of this data leaves your device. It is never transmitted to any external server, third party, or analytics service.

Data we do not collect

  • We do not collect personal information.
  • We do not use analytics or tracking tools.
  • We do not share any data with third parties.
  • We do not send any data to servers other than the Eventmaker API you are already using.

Permissions used

  • storage — to persist local cache (IndexedDB / localStorage) as described above.
  • host permissions for Eventmaker domains — to inject the overlay into Eventmaker back-office pages and to make GET requests to the Eventmaker API on your behalf.

Contact

If you have any questions about this policy, please contact the extension maintainer at your organization.

Politique de confidentialité

Eventmaker Atlas Overlay · Dernière mise à jour : 20 avril 2026

Présentation

Eventmaker Atlas Overlay est une extension Chrome qui affiche une surcouche de documentation de configuration sur le back-office Eventmaker. Cette page explique quelles données l'extension accède, comment elles sont stockées, et ce qu'elle ne fait pas.

Données accédées

L'extension accède aux données suivantes :

  • Données de configuration Eventmaker — l'extension effectue des requêtes GET en lecture seule vers l'API Eventmaker (la plateforme à laquelle vous êtes déjà connecté) pour récupérer des objets de configuration tels que les champs invités et les segments. Aucune credential n'est collectée ni stockée par l'extension ; elle s'appuie uniquement sur la session d'authentification déjà active dans votre navigateur.
  • URL de la page courante — l'extension lit l'URL de la page Eventmaker active pour détecter le contexte (quel événement, quelle section) et afficher les informations pertinentes.

Stockage des données

Toutes les données sont stockées localement sur votre appareil uniquement :

  • IndexedDB — utilisé pour mettre en cache les données de configuration Eventmaker récupérées depuis l'API.
  • localStorage — utilisé pour stocker des données légères telles que les événements récemment visités et les compteurs de visite par feature.

Ces données ne quittent jamais votre appareil. Elles ne sont jamais transmises à un serveur externe, à un tiers, ni à un service d'analyse.

Données non collectées

  • Nous ne collectons pas d'informations personnelles.
  • Nous n'utilisons pas d'outils d'analyse ou de tracking.
  • Nous ne partageons aucune donnée avec des tiers.
  • Nous n'envoyons aucune donnée vers des serveurs autres que l'API Eventmaker que vous utilisez déjà.

Permissions utilisées

  • storage — pour persister le cache local (IndexedDB / localStorage).
  • Permissions hôte sur les domaines Eventmaker — pour injecter la surcouche dans les pages du back-office Eventmaker et effectuer des requêtes GET vers l'API Eventmaker en votre nom.

Contact

Pour toute question relative à cette politique, contactez le mainteneur de l'extension au sein de votre organisation.
